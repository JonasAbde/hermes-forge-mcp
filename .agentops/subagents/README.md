# Subagents — forge-mcp

Dedicated subagents for common forge-mcp development workflows. Each subagent handles a specific domain with its own tools and context.

| Subagent | Focus |
|----------|-------|
| `discovery` | MCP tool discovery and search features |
| `evolution` | Agent fusion/synthesis tooling |
| `resilience` | Error handling, retries, circuit breaker |
| `http-server` | HTTP+SSE transport and Express server |
| `test-qa` | Test coverage, smoke tests, integration tests |
| `security-audit` | Security scanning and MCP-specific threats |
| `release-manager` | npm release pipeline and CI/CD |
| `docs-writer` | Documentation generation and maintenance |
