# MCP Configurations — forge-mcp

MCP server configurations for tool/service access. These define how forge-mcp connects to external MCP servers and services.

| Config | Purpose |
|--------|---------|
| `forge-api.json` | MCP connection to the main Hermes Forge API |
| `github.json` | GitHub API access for issue/PR management |
| `npm.json` | npm registry access for package management |
| `filesystem.json` | Local filesystem access for scripting |
| `web-search.json` | Web search for research and troubleshooting |
